1.面试题：大型页面时CSS的章法？
 一个大型页面（例如：京东首页）CSS大约有1000行，编写时有一定的章法：
 1.Reset:样式重置，把HTML标签的默认样式统一化——————可以使用Normalize.css
 2.Scaffolding:样式脚手架，设定全局的基础样式
 3.Layout System:布局系统
 4.Components:组件，多个页面所共有的通用组件必需的样式
 
 项目中大量小图标的处理方式
 方式1：每个小图标都是一个独立的.png文件，例如：search.png、user.png、star.png....HTML中使用<img>显示图片。
 不足：会增加浏览器资源请求次数，导致页面加载效率变低。
 方式2：把多个小图片拼接成一个大的.png图片（称为“Sprites Image”(精灵图/雪碧图)），使用时利用浏览器的缓存机制
 （请求一次之后，再次用到改图片无需再请求）可以实现大图片仅请求一次的效果
 不足：如果需要添加新的图标必须修改png图片；因为使用时必须配合CSS Background属性来使用，有些场合下不允许
 方式3.使用图标字体（icon font），本质是文字符（可以无限缩放/改变颜色/改变背景/加粗/变斜），呈现的外观是图标
 推荐使用开源的FontAwsome图标字体（提供了675个常用的图标）
 
 FontAwsome的使用步骤：
 1.下载字体文件，保存到项目目录下，如fonts/，以供用户浏览网页时可以下载到字体文件
 2.在CSS文件中声明两个class：
 @font-face{
	 font-family:'FontAwsome';
	 src:url('../fonts/FontAwsome.ttf');
 }
 .fa{
	 font-family:'FontAwsome';
 }
 .fa-bank:before{
	 content:'\f19c';//生成一个图标的编码
 }
 3.页面中引入上述CSS，开始使用图标字体：
 <span class='fa' fa-star>\f19c</span>
 注意：因为使用前置内容生成创建的图标，所以每个span中只能放置一个图标，即需要N个span
